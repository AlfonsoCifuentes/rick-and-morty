# Rick and Morty

> SPA Optimizada con Lazy Loading y Arquitectura Modular

## Resumen

Una Single Page Application (SPA) avanzada dedicada a Rick and Morty que implementa lazy loading y optimizaciones de rendimiento. La aplicación utiliza la API oficial de Rick and Morty para proporcionar una experiencia fluida de exploración de personajes, episodios y locaciones con técnicas modernas de optimización web.

## Valor del proyecto

Demuestra expertise en optimización de rendimiento web, arquitecturas SPA escalables, y técnicas avanzadas de carga diferida para aplicaciones de gran volumen de contenido.

Este repositorio resume una pieza de producto pensada para explicar el problema, la solucion aplicada y el criterio tecnico detras de la implementacion.

## Funcionalidades o puntos fuertes

- Arquitectura SPA con lazy loading implementado
- Integración completa con Rick and Morty API
- Sistema de routing optimizado para rendimiento
- Carga diferida de módulos y componentes
- Interface responsive y optimizada para móviles
- Sistema de búsqueda y filtrado avanzado
- Gestión eficiente de memoria y recursos
